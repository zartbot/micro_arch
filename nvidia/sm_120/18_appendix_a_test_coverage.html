Appendix A — Test Coverage

This appendix presents the complete test coverage matrix for the SM_120 (GB202) microbenchmark campaign. The campaign comprised 24 measurement sections (Sections 0-23) with **1504 individual test cases** in the core campaign, plus **136 follow-up cases** in Sections 24-26 (31 + 38 + 67 OQ-resolution experiments) for a total of **1640 cases**. Coverage statistics, pass/fail/anomaly distributions, and per-section data contribution mappings are provided for audit and reproducibility purposes.

A.6 Reproducibility Statement

All 1504 test cases are fully reproducible given:

  1. Access to an RTX PRO 5000 72GB (or equivalent SM_120 GB202 GPU)
  2. CUDA Toolkit 12.8 or later with sm_120a compilation support
  3. Fixed GPU clocks via nvidia-smi -lgc 2572,2572
  4. Exclusive GPU access (no concurrent processes)
  5. Source code in test_results/SM_120/section_{N}/src/
  6. Compilation flags per agent_docs/server.md

Measurement variance: <0.5% stddev for cycle-precise tests, <2% for throughput tests, <5% for power measurements.
